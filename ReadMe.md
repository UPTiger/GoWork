https://github.com/nutzam/ztcp
一个发送 TCP 命令的工具
本项目依赖

zgo : 需要它提供的一些便利函数
iconv-go : 需要用它来转换各种编码的 HTTP 页面
它们的安装很简单，请到它们的主页 查看其说明文档。我反正是这么装的:

# 安装 zgo
go get github.com/nutzam/zgo

# 安装 iconv-go
go get github.com/djimenez/iconv-go
